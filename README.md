# DStart

一个简洁轻巧的起始页

## 项目特点

- 纯 JavaScript 编写，无框架，结构清晰，性能良好
- 代码完全开源可读，亦可二次修改作为个人使用
- 默认不保存任何私人数据，当然你也可以选择共享它
- 支持拖拽和编辑导航项
- 支持多种方式自定义壁纸
- 支持导入导出个人设置（不支持导出壁纸图片）
